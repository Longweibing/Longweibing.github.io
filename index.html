Tersorflow学习笔记-1

Tensorflow是一种通过计算图构建神经网络、深度学习等模型，并用于实现相关任务的的工具。它主要由python语言实现，内部计算用C++实现，所以它的特点为：简单、高效、稳定。通过使用tensorflow不仅可以实现我们常见的深度学习任务，而且让我们从复杂的实现细节中解放出来，更加关注我们的任务本身，从而节省了宝贵的时间。本文主要是一些tensorflow的学习笔记，供我或者大家参考，具体细节还是看tensorflow官方文档更好。

September 1, 2018 Weibing Long

TENSORFLOW is a tool to build neural network, deep learning and other models by calculating graphs, and to realize related tasks. It is mainly implemented in PYTHON language, and internal computing is implemented in C++, so it is characterized as simple, efficient and stable. By using TENSORFLOW, we can not only achieve our common deep learning tasks, but also free us from the complex implementation details and pay more attention to our tasks, thus saving valuable time. This paper is mainly some study notes of TENSORFLOW for my or your reference. It is better to read the official documents of TENSORFLOW for details.
